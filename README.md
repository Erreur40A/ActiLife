# ActiLife

## Description

ActiLife est une application de suivi de santé développée dans le cadre du module Conduite
et Gestion des projets. Elle présente plusieurs options pour les utilisateurs: suivi de poids, de
sommeil, de calories, d’activités sportives, rappel hydratation, prise de médicaments.

Pour le rapport complet suivez ce [lien.](https://docs.google.com/document/d/1429y_Xl59l6GbcaUb2ymf_cYMQljtW3WtWkceTRbvJA/edit?usp=sharing)

## Technologies

ActiLife a été développée en Java. La base de données SQLite est intégrée à l'application.
Les interfaces ont été faites au moyen du fichier XML.
